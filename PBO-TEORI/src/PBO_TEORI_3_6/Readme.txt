Penjelasan tentang project 3.6.1 


Pada project tersebut kita membuat sebuah class. Class yang kita buat diberi nama kita sendiri.Pada kesempatan ini saya membuat class dengan nama "Muhammad_Irsyad_Al_Khairy".

Dalam penulisan class.nama file class harus sesuai dengan nama class di source code.


//dibawah adalah contoh penulisan kodenya.

public class Muhammad_Irsyad_Al_Khairy {
    
    public static void main(String[] args) {
        System.out.println("Welcome to Java Programming Muhammad Irsyad Al Khairy");
    }
    
}

Note : 1. public class Muhammad_Irsyad_Al_Khairy{} => untuk membuat class dengan nama "Muhammad_Irsyad_Al_Khairy"
     
     : 2. public static void main(String[] args){} => kode ini menjadi kode yang pertama di jalankan pada program java.
     
     : 3. System.out.println("Welcome to Java Programming Muhammad Irsyad Al Khairy"); => kode ini digunakan untuk menampilkan kode ke console/terminal.
     
     
output program => Welcome to Java Programming Muhammad Irsyad Al Khairy     
     
     
==============================================================================================================     


Penjelasan tentang project 3.6.2

Pada project tersebut kita membuat sebuah class.Saya membuat class dengan nama "TheTree".

Dalam penulisan file class penggunaan karakter special tidak diperbolehkan kecuali undescore ("_").



//dibawah adalah contoh penulisan kodenya.

public class TheTree {
    
    public static void main(String[] args) {
        System.out.println("I think that I shall never see,");
        System.out.println("a poem as lovely as a tree.");
        System.out.println("A tree whose hungry mouth is pressed");
        System.out.println("Against the Earth’s sweet flowing breast.");
    }
}

Note : 1. public class TheTree => pada kode ini kita membuat class dengan nama "TheTree"
       2. public static void main(String[] args){} => kode ini menjadi kode yang pertama di jalankan pada program java.
       3.  System.out.println("I think that I shall never see,");
           System.out.println("a poem as lovely as a tree.");
           System.out.println("A tree whose hungry mouth is pressed");
           System.out.println("Against the Earth’s sweet flowing breast.");
           
           => System.out.println(""); => digunakan untuk menampilkan program  ke console/terminal dengan 		
                                        membuat baris baru. 
           => System.out.print("");  => digunakan untuk menampilkan program  ke console/terminal tanpa 		
                                        membuat baris baru. 
      


output program =>   I think that I shall never see,
		    a poem as lovely as a tree.
	            A tree whose hungry mouth is pressed
	            Against the Earth’s sweet flowing breast.
